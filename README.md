# 顺其自然
 ## 前言
  成功和失败之间其实还有一条路可以走。
  
  只是当前国内的内卷环境让更多的年轻人失去了勇气去探寻这条平凡却又平静的道路。
  
  教育资源的短缺与不平衡，心理健康的忽视与妖魔化，资本的压榨与剥削，房价的荒唐与疯狂。。。
  
  归根到底，这不是中国年轻一代的过错。
  
  望你顺其自然，归来仍是追梦少年。

  ## 关于该项目
  纯粹是想让更多的人能够关注内卷的现状。毕竟年轻人是最无辜的群体。
  
  虽然以上的数据过少，并不能反映一个客观的现实。
  
  但是我希望正在看这个项目的你知道，我们所知道的只是冰山一角。
  
  因为初中高中生的消息往往直接会被压制住，毕竟这类学生群体也很少通过互联网来传播消息。
  
  同时，根据院校的水平，在网络上的舆论传播速度和效果也出现了不同级别的分化。
  
  越top的学校，公关手段会更加高明一些。比如这份名单里没出现全国压力最大的两所院校的数据。
  
  985/211等学校，因为名头响亮，在网上引起的轰动也会更为明显。
  
  而双非及以下的院校，出现类似的事情会少一点，所以更会被舆论忽视掉。
  
  但无论何种情况，院校都会想尽一切方法去压制这些新闻，仿佛一切都没发生过一样，然后下一次类似的事情又会像雨后春笋一样出现。恶性循环。
  
  而该项目，只是为了让更多的人知道，不该被遗忘的事情不会被遗忘。
